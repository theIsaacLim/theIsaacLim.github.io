Learn the Basics of Website Building Now!

26/5/2018

This is a quick guide to how to make a website

  1. Get a text editor for code

    • A text editor is a platform for writing code. I’ll be use brackets(brackets.io). It does syntax highlighting, which is highlighting correct bits of HTML, code completion and more.

  2. Write the HTML

    • HTML is the content of your page. It’s comprised of things called tags, which have a structure a bit like a burger. You have a piece of bread, the big shit, and then another piece of bread. HTML has an opening tag, a closing tag and then a bunch of big shit in the middle. Most of the time, this big shit will usually be a bundle of small burgers.

    • HTML tags also have things called attributes. They specify little things about the HTML like the source for an image or the link to link to for an anchor tag

    • There are different HTML tags for different things. Here are some important ones:

      3. Element name

      Symbol

      Purpose

      Paragraph tag

      <p>

      A new paragraph, the main font size that most text is written in

      Header tags

      <h1>, <h2>, <h3>, … <h6>

      Descending font sizes for headers/titles. h1 is biggest, h2 is second, h6 is smallest.

      Div tags

      <div>

      This is a container for things, usually used for readable CSS selectors or grouping things up

      Link tag

      <link>

      Links to a different file ie. the CSS

      Anchor tag

      <a>

      A link to another site, like the type the user clicks on

      Meta tag

      <meta>

      Tags about the document i.e. text encoding, screen viewport and more

      Head tag

      <head>

      A container for the head of the HTML, usually used to store meta tags and link tags

      Body tag

      <body>

      A container for the body of the HTML, used to store the main content of the webpage

      Image tag

      <img>

      Images, need src(source) attribute

  1. Write the CSS

    • CSS is the style of your page. It’s comprised of things called selectors and properties

    • Selectors select what you want to apply styles to. For example, to select all the paragraph tags you use the selector p

    • Properties describe how you want the elements you’ve selected with the selectors to be styled. For example, to change the text size, use the font-size property
